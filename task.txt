Задание:

Открытое API: https://reqres.in/
Свагер: https://reqres.in/api-docs/

На Python + PyTest написать тестовый фреймворк, где реализовать следующие пункты:
1) Написать позитивные и негативные API тесты, которые представлены на главной странице как образец
2) Написать WEB тесты (главная страница + свагер). Для свагера добавить проверку, что при нажатии на кнопку отправки образца запроса, получаемый результат (тело ответа и статус код) такой же как и через аналогичный API запрос.
3) Все тесты параметризировать и добавить фикстуры
4) Добавить возможность масштабировать проект (К примеру: если в WEB - добавится новая страница, а в API добавится новая версия API. То в таком случае добавляется новый класс и не нарушается текущая реализация)

Достаточно реализовать работу где-то с 5-тю api
Получается для API: 5-ть положительных кейсов, 5-ть негативных кейсов
Для WEB: 2 страницы по 5-ть тестов, где для кейсов свагера использовать сравнение с API
